# Demo Video Instructions

## Video File Location
The demo video should be uploaded as `demo-video.mp4` in the root directory of this repository.

## Video Content Requirements

The demo video should demonstrate the following key features:

### 1. Wallet Connection (0:00 - 0:30)
- Open the live demo at https://fitness-kappa-dusky.vercel.app/
- Click "Connect Wallet"
- Select Rainbow wallet
- Approve wallet connection
- Show successful connection status

### 2. Network Configuration (0:30 - 0:45)
- Switch to Sepolia testnet in wallet
- Verify contract detection
- Show network status in UI

### 3. Recording Training Data (0:45 - 1:30)
- Enter weight: 100 kg
- Enter sets: 3
- Enter reps: 10
- Click "Record Training Session"
- Show encryption progress
- Display transaction confirmation
- Show success message

### 4. Viewing Encrypted Records (1:30 - 2:00)
- Navigate to training history section
- Show encrypted records display
- Explain that data is stored encrypted on blockchain
- Refresh records to show latest data

### 5. Decrypting Records (2:00 - 2:45)
- Click "Decrypt" on a record
- Show decryption progress
- Display decrypted values (Weight: 100kg, Sets: 3, Reps: 10)
- Explain privacy-preserving nature

### 6. Error Handling Demo (2:45 - 3:15)
- Demonstrate form validation
- Show error messages for invalid inputs
- Handle network errors gracefully

### 7. Mobile Responsiveness (3:15 - 3:30)
- Show app working on mobile device
- Demonstrate responsive design

## Technical Specifications
- **Duration**: 3-5 minutes
- **Resolution**: 1920x1080 (Full HD)
- **Format**: MP4
- **Frame Rate**: 30 FPS
- **Audio**: Include voiceover explaining features

## Narration Script
Include clear voiceover explaining:
- Privacy-preserving encryption
- Zero-knowledge principles
- Real-world fitness use cases
- Technical implementation details
